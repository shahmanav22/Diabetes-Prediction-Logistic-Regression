# Diabetes Prediction using Logistic Regression

## Overview

This project applies **Logistic Regression** to predict whether a patient has diabetes based on medical and demographic information. It demonstrates a complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature encoding, correlation analysis, model training, and model evaluation.

---

## Dataset

The dataset used in this project is the **Diabetes Prediction Dataset** by Mohammed Mustafa on Kaggle.

Dataset Link:
https://www.kaggle.com/datasets/iammustafatz/diabetes-prediction-dataset

---

## Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Correlation Heatmap
- Label Encoding
- Feature Selection
- Train-Test Split
- Logistic Regression Model
- Model Evaluation

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Workflow

1. Import libraries
2. Load dataset
3. Explore and preprocess data
4. Encode categorical features
5. Visualize correlations using a heatmap
6. Train a Logistic Regression model
7. Evaluate model performance
